## 📖 개요

**환상의 세계 MUD**는 Python의 Textual 프레임워크를 사용하여 개발된 텍스트 기반 RPG 게임입니다. 기존의 웹 기반 버전을 완전히 리팩토링하여 터미널에서 실행되는 현대적인 TUI(Text User Interface) 게임으로 재탄생했습니다.

### 🎯 주요 특징

- **🎮 완전한 텍스트 기반 인터페이스**: 터미널에서 실행되는 현대적인 TUI
- **⚔️ 실시간 전투 시스템**: 몬스터와의 전략적 전투
- **🗺️ 탐험 가능한 월드**: 6개의 독특한 지역과 몬스터
- **📦 아이템 시스템**: 무기, 방어구, 물약, 보물 수집
- **📈 레벨업 시스템**: 경험치를 통한 캐릭터 성장
- **💰 경제 시스템**: 골드를 통한 아이템 거래
- **🎨 아름다운 UI**: Rich 라이브러리를 활용한 색상과 스타일링

## 🚀 설치 및 실행

### 필수 요구사항

- Python 3.8 이상
- pip (Python 패키지 관리자)

### 설치 방법

1. **저장소 클론**
   ```bash
   git clone <repository-url>
   cd mud-game
   ```

2. **의존성 설치**
   ```bash
   pip install -r requirements.txt
   ```

3. **게임 실행**
   ```bash
   python mud_game.py
   ```

## 🎮 게임 플레이

### 기본 조작법

| 명령어 | 설명 | 단축키 |
|--------|------|--------|
| `북`, `n`, `north` | 북쪽으로 이동 | `n` |
| `남`, `s`, `south` | 남쪽으로 이동 | `s` |
| `동`, `e`, `east` | 동쪽으로 이동 | `e` |
| `서`, `w`, `west` | 서쪽으로 이동 | `w` |
| `공격`, `attack`, `a` | 몬스터 공격 | `a` |
| `도망`, `flee`, `f` | 전투에서 도망 | `f` |
| `주변`, `look`, `l` | 주변 탐색 | `l` |
| `줍기`, `get`, `g` | 아이템 줍기 | `g` |
| `인벤토리`, `inventory`, `i` | 인벤토리 확인 | `i` |
| `사용`, `use`, `u` | 아이템 사용 | `u` |
| `도움말`, `help`, `h` | 도움말 표시 | `h` |
| `종료`, `quit`, `q` | 게임 종료 | `q` |

### 게임 목표

1. **탐험**: 6개의 다양한 지역을 탐험하세요
2. **전투**: 몬스터를 물리쳐 경험치와 골드를 획득하세요
3. **성장**: 레벨업하여 더 강한 캐릭터가 되세요
4. **수집**: 강력한 무기와 방어구를 수집하세요
5. **도전**: 최종 보스인 드래곤을 물리치세요!

## ⚔️ 전투 시스템

### 몬스터 정보

| 몬스터 | HP | 공격력 | 방어력 | 경험치 | 골드 | 설명 |
|--------|----|--------|--------|--------|------|------|
| 🧌 고블린 | 30 | 8 | 2 | 20 | 15 | 작고 교활한 고블린 |
| 👹 오크 | 60 | 15 | 5 | 40 | 30 | 강력한 오크 전사 |
| 🧟 트롤 | 100 | 25 | 10 | 80 | 60 | 거대하고 무서운 트롤 |
| 🐉 드래곤 | 200 | 40 | 20 | 200 | 150 | 전설의 드래곤 |

### 전투 메커니즘

1. **자동 조우**: 몬스터가 있는 방에 진입하면 자동으로 전투 시작
2. **턴제 전투**: 플레이어 공격 → 몬스터 반격 순서
3. **데미지 계산**: `데미지 = 공격력 - 방어력` (최소 1)
4. **도망 시스템**: 70% 확률로 전투에서 도망 가능
5. **보상**: 몬스터 처치 시 경험치와 골드 획득

## 📦 아이템 시스템

### 아이템 종류

| 아이템 | 타입 | 효과 | 가치 | 설명 |
|--------|------|------|------|------|
| 🗡️ 녹슨 검 | 무기 | 공격력 +5 | 10 | 오래된 검 |
| ⚔️ 철검 | 무기 | 공격력 +15 | 50 | 견고한 철검 |
| 🛡️ 가죽 갑옷 | 방어구 | 방어력 +3 | 20 | 가벼운 가죽 갑옷 |

| 🪙 금화 | 보물 | 골드 +100 | 100 | 빛나는 금화 |

### 아이템 사용법

- **무기/방어구**: 자동으로 장착되어 스탯 증가
- **물약**: 사용 시 체력 회복
- **보물**: 골드로 자동 변환

## 🎯 게임 전략

### 초보자 가이드

1. **시작**: 시작의 마을에서 체력 물약을 획득하세요
2. **탐험**: 북쪽 숲으로 이동하여 고블린과 전투하세요
3. **장비**: 녹슨 검을 획득하여 공격력을 높이세요
4. **성장**: 경험치를 쌓아 레벨업하세요
5. **준비**: 충분히 강해진 후 어두운 동굴에 도전하세요

### 고급 전략

1. **효율적인 전투**: 체력이 낮으면 도망을 선택하세요
2. **아이템 관리**: 물약은 위급할 때만 사용하세요
3. **경로 최적화**: 안전한 지역을 통해 이동하세요
4. **최종 도전**: 드래곤은 최소 레벨 5 이상에서 도전하세요


